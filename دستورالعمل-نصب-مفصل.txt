📁 مرحله دوم: آپلود فایل‌های نصب
═════════════════════════════════════════════════════════════════

پس از ورود به کنترل پنل:

1. از فهرست سمت چپ، "File Manager" یا "مدیریت فایل" را انتخاب کنید

2. پوشه "public_html" را دوبار کلیک کنید

3. فایل‌های زیر را آپلود کنید:
   ┌─────────────────────────────────────────────────────────┐
   │ فایل                          │ اندازه   │ اولویت      │
   ├─────────────────────────────────────────────────────────┤
   │ setup-final.php              │ ~30 KB  │ ⭐⭐⭐      │
   │ fruit-shop-products.csv      │ ~2 KB   │ ⭐⭐⭐      │
   │ (سایر فایل‌های HTML/CSS)      │ متفاوت │ ⭐ (اختیاری) │
   └─────────────────────────────────────────────────────────┘

4. برای آپلود:
   • روی دکمه "Upload" کلیک کنید
   • فایل‌های مورد نظر را انتخاب کنید
   • منتظر بمانید تا آپلود کامل شود

⏱️  مدت زمان آپلود: 1-2 دقیقه

🚀 مرحله سوم: اجرای اسکریپت نصب
═════════════════════════════════════════════════════════════════

1. بعد از آپلود موفق، مرورگر یک تب جدید باز کنید

2. به آدرس زیر بروید:
   👉 https://mivesf.ir/setup-final.php

3. صبر کنید تا اسکریپت اجرا شود (مدت زمان: 2-5 دقیقه)

4.🎯 اسکریپت مراحل زیر را انجام می‌دهد:
   ✅ بررسی WooCommerce
   ✅ ایجاد صفحات ضروری (فروشگاه، سبد خرید، پرداخت، حساب کاربری)
   ✅ ایجاد دسته‌بندی‌های محصول (5 دسته)
   ✅ آپلود 19 محصول از فایل CSV
   ✅ تنظیمات WooCommerce (واحد پول: تومان، زبان: فارسی)

5. بررسی خروجی:
   • اگر سبز است (✅): موفق
   • اگر نارنجی است (⚠️): هشدار (قبلاً موجود)
   • اگر قرمز است (❌): خطا (نیاز به حل)

     - واحد پول: تومان (IRR)

🛒 مرحله پنجم: بازدید فروشگاه
═════════════════════════════════════════════════════════════════

1. مرورگر یک تب جدید باز کنید

2. به آدرس زیر بروید:

3. بررسی نکات زیر:
   ✅ محصولات در فروشگاه نمایش داده می‌شوند
   ✅ قیمت‌ها به صورت صحیح نمایش داده می‌شوند
   ✅ دسته‌بندی‌های محصول دیده می‌شود
   ✅ زبان صفحه فارسی است
   ✅ طراحی (Theme) زیبا و حرفه‌ای است


═════════════════════════════════════════════════════════════════
⚙️ مرحله ششم: تنظیمات روش‌های پرداخت (اختیاری)
═════════════════════════════════════════════════════════════════

برای فعال کردن روش‌های پرداخت:

1. داشبورد > WooCommerce > تنظیمات > پرداخت بروید

2. روش‌های پرداخت مورد نظر را فعال کنید:
   ✓ درگاه پرداخت الکترونیکی
   ✓ انتقال بانکی
   ✓ پرداخت در محل
   (بستگی به سرویس‌های موجود هاست شما دارد)

3. هر روش را تنظیم کنید

4. تغییرات را ذخیره کنید


═════════════════════════════════════════════════════════════════
📧 مرحله هفتم: تنظیم ایمیل (اختیاری)
═════════════════════════════════════════════════════════════════

برای دریافت ایمیل‌های سفارش:

1. داشبورد > تنظیمات > عمومی بروید

2. ایمیل مدیریت:
   • ایمیل: admin@mivesf.ir (استفاده کنید)

3. داشبورد > WooCommerce > تنظیمات > ایمیل‌ها بروید

4. تنظیمات ایمیل‌های سفارش، پیام جدید و غیره


═════════════════════════════════════════════════════════════════
🔐 مرحله هشتم: امنیت (اختیاری اما توصیه شده)
═════════════════════════════════════════════════════════════════

برای امنیت بیشتر:

1. فایل setup-final.php را حذف کنید
   (از طریق File Manager یا FTP)

2. رمز عبور مدیر WordPress را تغییر دهید:
   داشبورد > کاربران > (نام کاربری) > ویرایش

3. تنظیم SSL (HTTPS):
   • کنترل پنل > SSL/TLS
   • اگر موجود است، فعال کنید
